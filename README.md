## рџ“– Project Overview

The **Number Guessing Game** is a Python-based console application that challenges users to guess a randomly generated number within a limited number of attempts. The program provides real-time feedback indicating whether the user's guess is too high or too low, making the game interactive and engaging.

This project demonstrates the use of Python fundamentals such as loops, conditional statements, exception handling, user input validation, and random number generation.

## рџЋЇ Objectives

* Generate a random number between 1 and 100.

* End the game when the user guesses correctly or reaches the maximum number of attempts.

## рџЋ® How to Play

1. Start the program.
2. The computer randomly selects a number between 1 and 100.
3. Enter your guess when prompted.
4. The game will display:

   * **Too High** if your guess is greater than the secret number.
   * **Too Low** if your guess is smaller than the secret number.
5. Continue guessing until:

   * You find the correct number, or
   * You run out of attempts.
